Performs a CREATE KEY operation, which creates a key and stores in the service.

Name Type Attributes Description

The vault identifier.

The key name. If a key with this name already exists, the service creates a new version and set as current.

The key type. Use a constant from JsonWebKeyType.

The key size. Must be valid for the key type. If omitted, a default for the key type is used.
