SqlManagementService

The SqlManagementService object allows you to perform managment operations on Microsoft Azure SQL Database Servers.

Constructor

new SqlManagementService(configOrSubscriptionId, authentication, hostOptionsopt)

Creates a new SqlManagementService object

Parameters:
Name Type Attributes Description
configOrSubscriptionId string

Configuration, subscription ID for the account or the connection string
authentication object

The authentication object for the client.
You must use either keyfile/certfile or keyvalue/certvalue
to provide a management certificate to authenticate
service requests.

Properties
Name Type Attributes Description

The path to a file that contains the PEM encoded key

certvalue string <optional>
