ApnsService

ApnsService

The ApnsService class is used to send notifications using the Apple Push Notification Service (APNS).
This service is used to communicate with iOS device.

Constructor

new ApnsService(notificationHubService)

Creates a new ApnsService object using the specified NotificationHubService object.

Note: An instance of this object is created automatically when a NotificationHubService
object is created. See NotificationHubService#apns.

Parameters:
Name Type Description
notificationHubService NotificationHubService

The notification hub service.

send(tags, payload, callback)

Sends an APNS notification.

Parameters:
Name Type Description
tags string

A single tag or tag expression.
If null it will broadcast to all registrations in this hub
payload object | string

The message's JSON payload as specified below.
If the payload is a string, follow the APNS format as in https://developer.apple.com/library/ios/documentation/NetworkingInternet/Conceptual/RemoteNotificationsPG/Chapters/ApplePushService.html#//apple_ref/doc/uid/TP40008194-CH100-SW1

Properties
Name Type Attributes Description
expiry date <optional>

The expiration date. (in W3C DTF, YYYY-MM-DDThh:mmTZD (e.g. 1997-07-16T19:20+01:00))
aps object <optional>

If the 'aps' member is provided, the object is delivered to APNS as-is after expiry is processed. The object is assumed to follow the APNS format at https://developer.apple.com/library/ios/documentation/NetworkingInternet/Conceptual/RemoteNotificationsPG/Chapters/ApplePushService.html#//apple_ref/doc/uid/TP40008194-CH100-SW1
badge number <optional>

If the 'aps' member is not provided, the number to display over the app icon.
alert string <optional>

If the 'aps' member is not provided, the alert text.
sound string <optional>

If the 'aps' member is not provided, the sound file name.
payload object <optional>

If the 'aps' member is not provided, the payload object that contains the notification text.
callback function

error will contain information
if an error occurs; otherwise, response
will contain information related to this operation.

  • apnsservice.js, line 73
Example
notificationHubService.apns.send( null, {
    alert: 'This is my toast message for iOS!',
    expiry: '2014-07-16T19:20+01:00'
  },
  function (error) {
    if (!error) {
      // message sent successfully
    }
});
