MpnsService

MpnsService

The MpnsService class is used to send notifications using the Microsoft Push Notification Service.

Constructor

new MpnsService(notificationHubService)

Creates a new MpnsService object.

Note: An instance of this object is created automatically when a NotificationHubService
object is created. See NotificationHubService#gcm.
