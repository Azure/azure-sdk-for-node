Class: BlobService

BlobService

The BlobService class is used to perform operations on the Microsoft Azure Blob Service. The Blob Service provides storage for binary large objects, and provides functions for working with data stored in blobs as either streams or pages of data.

For more information on the Blob Service, as well as task focused information on using it in a Node.js application, see How to Use the Blob Service from Node.js.
